# Easy easy search
# 带本地过滤表的综合文献搜索
本地模拟网页请求，对请求结果进行过滤筛查  
- 通过requests向不同论文网站发送搜索请求
- 通过PySide2绘制ui界面
- 本地存储文献过滤表，对搜索结果进行筛查，再进行显示

v1：2021-2月期间开发，能对nature进行查询并过滤本地论文标题列表
